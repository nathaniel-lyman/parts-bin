Searchable, grouped command launcher with keyboard shortcut.

**When to use:** A global, type-to-find action launcher across the app.

**Key props:** `groups`, `open`, `onOpenChange`, `enableGlobalShortcuts`, `placeholder`, `trigger`

**Example:**
```tsx
<CommandPalette groups={groups} open={open} onOpenChange={setOpen} />
```
