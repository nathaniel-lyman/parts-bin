Square icon-only button; same variants as Button, requires aria-label.

**When to use:** A toolbar or compact control where an icon stands in for a text label.

**Key props:** `variant`, `size`, `loading`, `aria-label`, `onClick`

**Example:**
```tsx
<IconButton aria-label="Edit" onClick={edit}><EditIcon /></IconButton>
```
