List of attachments with name, size, status, and remove action.

**When to use:** Displaying files attached to a record.

**Key props:** `attachments`, `emptyLabel`, `className`

**Example:**
```tsx
<AttachmentList attachments={[{ id: 'f1', name: 'report.pdf', size: 12000 }]} />
```
